compat_comments: Initial boot screen renders correctly. Has 3 options to select:
(1) NCAA Football 2005
(2) Top Spin
(3) EA Sports on Xbox Live

Selecting option 3 renders an FMV that plays well and returns to main menu upon completion.
Selecting option 1 or 2 will launch the respective game title.

NCAA Football 2005: Starts
The game has menu functionality and renders FMVs alright.
However, in a match there is Z-fighting on the field.
Frame rate drops to less than 10 FPS once a play begins, about 25-30 FPS in scenes.

Top Spin: Playable
All menus appear to run fine and as expected.
An exhibition match can be completed successfully, and without any issues.
Create-a-player mode however has many graphical glitches for the player's body and gear.
Sometimes the created player can render in shadow or corrupted textures, both in the menu or in a match.
